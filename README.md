HOW TO RUN

```bash
# pnpmがインストールされてない場合
# 注意：必ずnpmでしてください（yarnでしたら実行ファイルのPATHに追加されない）
npm i pnpm -g

# npm iと同義です
# クソ速い
pnpm i

# まだweb-ext入れてないのでdistに吐き出すだけ
pnpm build
```
